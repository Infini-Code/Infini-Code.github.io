Starting Your Project

To get started open Visual Studio Code and make a new project.

Now click the extensions button on the left of the screen then type "Live Server" and install.

Now we can edit our website locally and see the changes live!

Create a new file called "index.html" in your new project.

Ok, so now we can start making our site. Copy the HTML code below and paste into your index.html file.

<!DOCTYPE html>
<html lang="en">
<body>
<h1>Welcome to your new website!</h2>
</body>
</html>

Now right click your index.html file and click "open with live server".

Then if you did it correctly your browser should open with your new website!
